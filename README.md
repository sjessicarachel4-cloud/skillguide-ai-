
# 🎓 Student Guidance System (Prototype)

## 📌 Overview
The Student Guidance System is a **prototype project** that helps students explore suitable **courses and internships** based on their interests such as **AI, Data, Web, and Cloud**.  
The system analyzes user input using **keyword-based domain detection** and provides curated recommendations.

This project is intended as a **proof of concept** and focuses on demonstrating the core logic of an interest-driven recommendation system.

---

## 🎯 Features
- Accepts user interest in **any format** (e.g., AI, ai, artificial intelligence, ML)
- Automatically detects the relevant domain
- Recommends:
  - 📚 Online courses (with title, duration, language, and URL)
  - 💼 Internship opportunities (with role, location, experience, and skills)
- Displays up to **10 recommendations** per category
- Simple and beginner-friendly Python implementation

---

## 🛠 Technologies Used
- Python
- Pandas
- CSV datasets
- Command Line Interface (CLI)
